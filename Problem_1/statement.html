In this problem, there is three operations...

  • 1 X
  • 2 L, R, value
  • 3 X

In this opeartion..

  • In 1st Operation:

    If the index is not explored, then explore this index and set the value of this index as 0.

    If the index is explored, then just set the value of this index as 0.

  • In 2nd Operation:

    Add the value to all index between the range L to R. which are already explored.

  • In 3rd Operation:

    get the value of the index X

Input

The first line of input will contain Q (1<= Q <= 105) number of queries.

In the next Q line will content those three operation.

Constant

  • 1 X ; here (1<= X <= 106)
  • 2 L, R, value ; here (1<= L<=R <= 106) and (1<= val <= 109)
  • 3 X ; here (1<= X <= 106)

Output

Output only for qi = 3 operations.

print "Value" of the index.

Example

3rd query: 4 is exprolored and set the value as 0.
4th query: from range 3 to 5. we have only one index explored. which is 4. so, add 3 to the index 4.
5th query: 1 is exprolored and set the value as 0.
6th query: from range 1 to 3. we have two index explored. which is 1,2. so, add 2 to the index 1 and 2.
7th query: print the value of the index 1. which is 2.
8th query: print the value of the index 2. which is 7.
9th query: print the value of the index 3. which is 0 (it's not exprolored yet).
